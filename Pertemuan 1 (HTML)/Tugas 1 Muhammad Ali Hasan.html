Sejarah & Perkembangan HTML

Pada tahun 1980, fisikawan Tim Berners-Lee , seorang kontraktor di CERN , mengusulkan dan membuat prototipe INQUIRE , sebuah sistem bagi peneliti CERN untuk menggunakan dan berbagi dokumen. Pada tahun 1989, Berners-Lee menulis sebuah memo yang mengusulkan sistem hypertext berbasis Internet . Berners-Lee menentukan HTML dan menulis peranti lunak peramban dan server pada akhir 1990. Tahun itu, Berners-Lee dan insinyur sistem data CERN Robert Cailliau berkolaborasi dalam permintaan bersama untuk pendanaan, tetapi proyek itu tidak secara resmi diadopsi oleh CERN. Dalam catatan pribadinya dari tahun 1990 ia mendaftar "beberapa dari banyak area di mana hypertext digunakan" dan menempatkan ensiklopedia terlebih dahulu.
Deskripsi HTML pertama yang tersedia untuk umum adalah dokumen yang disebut "Tag HTML" , yang pertama kali disebutkan di Internet oleh Tim Berners-Lee pada akhir 1991. Ini menjelaskan 18 elemen yang terdiri dari desain awal HTML yang relatif sederhana. Kecuali untuk tag hyperlink, ini sangat dipengaruhi oleh SGMLguid , format dokumentasi berbasis Standard Generalized Markup Language (SGML) internal di CERN. Sebelas elemen ini masih ada di HTML 4.
HTML adalah bahasa markup yang digunakan browser web untuk menafsirkan dan menulis teks, gambar, dan materi lainnya ke dalam halaman web visual atau terdengar. Karakteristik default untuk setiap item markup HTML ditentukan di browser, dan karakteristik ini dapat diubah atau ditingkatkan dengan penggunaan CSS tambahan oleh desainer halaman web. Banyak elemen teks ditemukan dalam laporan teknis ISO 1988 TR 9537 Teknik untuk menggunakan SGML , yang pada gilirannya mencakup fitur bahasa pemformatan teks awal seperti yang digunakan oleh perintah RUNOFF yang dikembangkan pada awal 1960-an untuk CTSS (Waktu yang Kompatibel -Sharing System) sistem operasi: perintah pemformatan ini berasal dari perintah yang digunakan oleh pembuat huruf untuk memformat dokumen secara manual. Namun, konsep SGML tentang markup umum didasarkan pada elemen (rentang beranotasi bersarang dengan atribut) daripada hanya efek cetak, dengan juga pemisahan struktur dan markup; HTML telah secara progresif dipindahkan ke arah ini dengan CSS.
Berners-Lee menganggap HTML sebagai aplikasi SGML. Itu secara formal didefinisikan oleh Internet Engineering Task Force (IETF) dengan publikasi proposal pertama pada pertengahan 1993 untuk spesifikasi HTML, Draft Internet "Hypertext Markup Language (HTML)" oleh Berners-Lee dan Dan Connolly , yang mana termasuk definisi tipe Dokumen SGML untuk mendefinisikan tata bahasa. Draf tersebut kedaluwarsa setelah enam bulan, tetapi terkenal karena pengakuannya atas tag kustom browser NCSA Mosaic untuk menyematkan gambar sebaris, yang mencerminkan filosofi IETF dalam mendasarkan standar pada prototipe yang berhasil. Demikian pula, Internet-Draft Dave Raggett yang bersaing, "HTML + (Hypertext Markup Format)", dari akhir 1993, menyarankan standarisasi fitur yang sudah diimplementasikan seperti tabel dan formulir isian.
Setelah draf HTML dan HTML + kedaluwarsa pada awal tahun 1994, IETF membuat Kelompok Kerja HTML, yang pada tahun 1995 menyelesaikan "HTML 2.0", spesifikasi HTML pertama yang dimaksudkan untuk diperlakukan sebagai standar yang menjadi dasar penerapan di masa mendatang.
Perkembangan lebih lanjut di bawah naungan IETF terhenti oleh persaingan kepentingan. Sejak tahun 1996, spesifikasi HTML telah dipertahankan, dengan masukan dari vendor perangkat lunak komersial, oleh World Wide Web Consortium (W3C). Namun, pada tahun 2000, HTML juga menjadi standar internasional ( ISO / IEC 15445: 2000). HTML 4.01 diterbitkan pada akhir 1999, dengan ralat lebih lanjut diterbitkan hingga 2001. Pada tahun 2004, pengembangan HTML5 dimulai di Kelompok Kerja Teknologi Aplikasi Hypertext Web (WHATWG), yang menjadi penyampaian bersama dengan W3C pada tahun 2008, dan diselesaikan serta distandarisasi pada 28 Oktober 2014.
berikut adalah perkembangan HTML :
Tahun 1995 tanggal 24 November, HTML 2.0 resmi dipublikasikan sebagai RFC 1866 yaitu seri dokumen informasi dengan standar internet yang bernomer.
Tahun 1997 tanggal 14 Januari, HTML 3.2 dipublikasikan dan menjadi versi yang distadardisasi dari versi sebelumnya, dikembangkan oleh Konsorisum World Wide Web.
Tahun 1997 tanggal 18 Desember, HTML 4.0, resmi dipublikasikan sebagai W3C yang rekomendasi, dimana memiliki tiga variasi yaitu bersifat Ketat, Transisional (elemen terprediksi) dan Frameset (elemen yang berkaitan dengan frame).
Tahun 2000, W3C mengembangkan XHTML 1.0.
Tahun 2008, HTML 5 resmi pertama kali dipublikasikan sebagai draft oleh WHATWG (Web Hypertext Application Technology Working Group) yaitu pada proyek yang bekerjasama dengan W3C, dan menjadi standar baru bagi HTML, XHTML dan DOM HTML.
Tahun 2012, HTML 5 resmi sebaga standar web oleh WHATWG.
Tahun 2014, HTML 5 menjadi markup yang telah sempurna dan rekomendasi.

HTML ver. 1.0

Bermula pada versi 1.0. Versi 1.0 ini merupakan pionir yang masih banyak kelemahan sehingga tampilannya masih sederhana. HTML versi 1.0 mempunyai kemampuan untuk membuat paragraf, heading, list, hypertext dan cetak tebal atau miring pada teks. Versi 1.0 juga dapat mendukung peletakan gambar atau image pada dokumen tanpa wrapping.

HTML ver. 2.0

HTML versi 2.0 mempunyai fitur tambahan yaitu kualitas HTML lebih baik. Versi 2.0 mempunyai kemampuan dalam menampilkan data atau form pada dokumen serta memasukan alamat, nama, dan saran atau kritik. HTML versi 2.0 ini adalah pionir untuk web interaktif.

HTML ver. 3.0

Versi 3.0 mempunyai tambahan fitur dengan fasilitas baru yaitu figure. Figure adalah perkembangan dari image yang berfungsi untuk meletakan gambar serta tabel. HTML versi 3.0 juga dapat mendukung adanya rumus matematika pada dokumen.

HTML ver. 4.0

HTML versi 4.0 mengalami banyak perubahan dari versi sebelumnya. Perubahan tersebut pada perintah HTML yaitu image, tabel, text, link, form dan sebagainya.

HTML ver. 5.0

HTML versi 5 merupakan prosedur pembuatan tampilan web terbaru dengan penggabungan antara CSS, HTML dan JavaScript. Versi ini dibuat dari informasi bahwa W3C dan IETF membuat versi HTML yang terbaru.
